Le style d'une ligne est définit par 6 styles de cellules. Le premier style de cellule est utilisé pour 
les colonnes avant la colonne 'Rubrique', le second pour la colonne'Rubrique', le troisième pour les 
colonnes entre 'Rubrique' et 'libelle', le quatrieme pour la colonne 'libelle', le cinquieme pour
les colonnes restantes sauf la dernière et le sixième pour la dernière colonne.


cellule :nomPolice;taillePolice;Gras;couleurPolice;couleurFond;bordureTop;bordureBotom;bodureLeft;bordureRight;typeCellule

couleurs : code RGB du type R153G204B255
bordures : Thick, Thin, ou absent
type de cellule :
		2 : nombre avec 2 chiffres apres la virgule
		3 : nombre avec espace pour séparateur des milliers, millions...
		10 : poucenage avec 2 chifres après la virgule
